# Engineering Pattern Library

This document will grow throughout the course.

| Problem | Pattern |
|----------|----------|
| Repeated Reads | Caching |
| Long Running Tasks | Queue |
| Duplicate Requests | Idempotency |
| Temporary Failures | Retry |
| Cascading Failures | Circuit Breaker |
| Traffic Spikes | Rate Limiting |
| Uneven Traffic Distribution | Load Balancing |
| Large Dataset | Partitioning |
| High Availability | Replication |
| Need Fast Search | Search Index |
| Distributed Transactions | Saga |
| Read/Write Separation | CQRS |
| Static Content Delivery | CDN |
| Horizontal Data Scaling | Sharding |

---

## Rule

Never start with technology.

Always identify the engineering pattern first.
